## クイックスタート

### インストール

```bash npm2yarn
npm install --save-dev prisma-zod-mock
```

### 基本的な使用方法

```prisma filename="schema.prisma"
generator zodMock {
  provider = "prisma-zod-mock"
  output   = "./generated"
}

model User {
  id        String   @id @default(cuid())
  email     String   @unique
  name      String?
  createdAt DateTime @default(now())
}
```

### モックデータの生成

```typescript filename="app.ts"
import { createUserMock } from './generated';

const user = createUserMock();
console.log(user);
// {
//   id: 'clh3k4n5j0000qw8w5h3k4n5j',
//   email: 'test@example.com',
//   name: 'John Doe',
//   createdAt: 2024-01-01T00:00:00.000Z
// }
```
