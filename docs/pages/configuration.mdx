# 設定

## ジェネレーターオプション

`prisma-zod-mock`は、`schema.prisma`ファイルのgeneratorブロックで様々なオプションを設定できます。

```prisma filename="schema.prisma"
generator zodMock {
  provider                = "prisma-zod-mock"
  output                 = "./generated"
  
  // オプション設定
  createZodSchemas       = true
  createMockFactories    = true
  useMultipleFiles       = false
  writeBarrelFiles       = true
  mockDataLocale        = "ja"
  mockDateRange         = 30
  createRelationMocks   = true
}
```

## 利用可能なオプション

### 基本設定

| オプション | デフォルト | 説明 |
|----------|---------|-----|
| `output` | `"./generated"` | 生成されたファイルの出力先ディレクトリ |
| `createZodSchemas` | `true` | Zodスキーマを生成するかどうか |
| `createMockFactories` | `true` | モックファクトリー関数を生成するかどうか |

### ファイル管理

| オプション | デフォルト | 説明 |
|----------|---------|-----|
| `useMultipleFiles` | `false` | モデルごとに個別のファイルを生成するかどうか |
| `writeBarrelFiles` | `true` | バレルファイル（index.ts）を生成するかどうか |

### モックデータ設定

| オプション | デフォルト | 説明 |
|----------|---------|-----|
| `mockDataLocale` | `"en"` | Faker.jsのロケール設定（例: `"ja"`, `"en"`, `"fr"`） |
| `mockDateRange` | `30` | 日付生成時の前後の日数範囲 |
| `createRelationMocks` | `true` | リレーションを含むモックファクトリーを生成するかどうか |

## 設定例

### 日本語データの生成

```prisma
generator zodMock {
  provider       = "prisma-zod-mock"
  mockDataLocale = "ja"
}
```

### マルチファイル出力

```prisma
generator zodMock {
  provider         = "prisma-zod-mock"
  output          = "./generated/mocks"
  useMultipleFiles = true
}
```

この設定により、以下のような構造でファイルが生成されます：

```
generated/mocks/
├── index.ts
├── user/
│   ├── user.schema.ts
│   └── user.mock.ts
└── post/
    ├── post.schema.ts
    └── post.mock.ts
```

### Zodスキーマのみ生成

```prisma
generator zodMock {
  provider            = "prisma-zod-mock"
  createZodSchemas    = true
  createMockFactories = false
}
```

## 環境変数での設定

一部の設定は環境変数でも指定できます：

```bash
# ロケール設定
PRISMA_ZOD_MOCK_LOCALE=ja

# 日付範囲
PRISMA_ZOD_MOCK_DATE_RANGE=60
```

環境変数は`schema.prisma`での設定より優先されます。
