## Quick Start

<Steps>

### Installation

Install with your favorite package manager:

<Tabs items={['npm', 'pnpm', 'yarn', 'bun']}>
  <Tab>```bash npm install --save-dev prisma-zod-mock ```</Tab>
  <Tab>```bash pnpm add -D prisma-zod-mock ```</Tab>
  <Tab>```bash yarn add -D prisma-zod-mock ```</Tab>
  <Tab>```bash bun add -d prisma-zod-mock ```</Tab>
</Tabs>

### Prisma Schema Setup

Add the generator to your `schema.prisma`:

```prisma filename="schema.prisma" {3-6}
datasource db {
  provider = "postgresql"
  url      = env("DATABASE_URL")
}

generator zodMock {
  provider = "prisma-zod-mock"
  output   = "./generated"
}

model User {
  id        String   @id @default(cuid())
  email     String   @unique
  name      String?
  createdAt DateTime @default(now())
}
```

### Generate Code

Run the Prisma generator:

```bash
npx prisma generate
```

### Usage

Create mock data using the generated code:

```typescript filename="app.ts"
import { createUserMock } from './generated';

const user = createUserMock();
console.log(user);
// {
//   id: 'clh3k4n5j0000qw8w5h3k4n5j',
//   email: 'test@example.com',
//   name: 'John Doe',
//   createdAt: 2024-01-01T00:00:00.000Z
// }
```

</Steps>

## Use Cases

### 🧪 Testing

Generate consistent mock data for unit and integration tests

```typescript
describe('UserService', () => {
  it('should create a user', async () => {
    const mockUser = createUserMock();
    const result = await userService.create(mockUser);
    expect(result).toEqual(mockUser);
  });
});
```

### 🌱 Seeding

Populate development and test databases with realistic data

```typescript
async function seed() {
  const users = Array.from({ length: 100 }).map(() => createUserMock());
  await prisma.user.createMany({ data: users });
}
```

### 📝 Type Validation

Validate API request/response types

```typescript
const userInput = req.body;
const validatedUser = UserSchema.parse(userInput);
```
