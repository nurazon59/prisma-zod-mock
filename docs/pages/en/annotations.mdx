# Annotations

Learn how to customize mock data using field-level annotations.

## Overview

Use annotations to fine-tune the behavior of generated mock data.

```prisma
model User {
  id    String @id @default(cuid())
  /// @mock faker.internet.email()
  email String @unique
  /// @mock faker.person.fullName()
  name  String?
}
```

## Faker.js Methods

Use any Faker.js method to generate data.

```prisma
model Product {
  /// @mock faker.commerce.productName()
  name        String
  /// @mock faker.commerce.productDescription()
  description String
  /// @mock faker.number.float({ min: 10, max: 1000, fractionDigits: 2 })
  price       Float
  /// @mock faker.image.url()
  imageUrl    String
}
```

## Fixed Values

Set fixed values for fields.

```prisma
model Config {
  id       String @id
  /// @mock "production"
  env      String
  /// @mock "v1.0.0"
  version  String
  /// @mock true
  enabled  Boolean
}
```

## Range Specification

Specify ranges for numeric fields.

```prisma
model Product {
  id    String @id
  /// @mock.range(1, 100)
  stock Int
  /// @mock.range(0.0, 5.0)
  rating Float
}
```

## Patterns

Generate data based on regex patterns.

```prisma
model Account {
  id           String @id
  /// @mock.pattern("[A-Z]{3}-[0-9]{4}")
  accountCode  String  // Generates: ABC-1234
  /// @mock.pattern("\\+81[0-9]{2}-[0-9]{4}-[0-9]{4}")
  phoneNumber  String  // Generates: +8190-1234-5678
}
```

## Enum Values

Randomly select from a list of specified values.

```prisma
model User {
  id     String @id
  /// @mock.enum("admin", "user", "guest")
  role   String
  /// @mock.enum("active", "inactive", "pending")
  status String
}
```

## Complex Examples

### Combining Annotations

```prisma
model Article {
  id          String   @id @default(cuid())
  /// @mock faker.lorem.sentence()
  title       String
  /// @mock faker.lorem.paragraphs(3)
  content     String
  /// @mock faker.helpers.arrayElement(["draft", "published", "archived"])
  status      String
  /// @mock faker.number.int({ min: 0, max: 10000 })
  views       Int      @default(0)
  /// @mock faker.date.recent({ days: 30 })
  publishedAt DateTime?
}
```

### User Profile Example

```prisma
model UserProfile {
  id          String  @id @default(cuid())
  /// @mock faker.internet.userName()
  username    String  @unique
  /// @mock faker.person.fullName()
  displayName String
  /// @mock faker.person.bio()
  bio         String?
  /// @mock faker.image.avatar()
  avatarUrl   String?
  /// @mock faker.date.birthdate({ min: 18, max: 80, mode: 'age' })
  birthDate   DateTime
  /// @mock faker.location.country()
  country     String
  /// @mock faker.helpers.arrayElement(["free", "pro", "enterprise"])
  plan        String  @default("free")
}
```
